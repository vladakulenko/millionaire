# 🎮 Who Wants to Be a Millionaire — Quiz Game

A quiz game built with **Next.js (App Router) + TypeScript**.

The player answers questions, progresses through prize levels, and sees the final result screen with total winnings.

---

## 🚀 Technologies Used

- Next.js (App Router)
- React
- TypeScript
- CSS Modules
- JSON-based game configuration

---

## 📦 Installation

### 1. Clone the repository

git clone https://github.com/vladakulenko/millionaire.git

### 2. Navigate to the project folder

cd .\millionaire\

### 3. Install dependencies

npm install

### 4. Run the project (Development mode)

npm run dev and go to http://localhost:3000

### 5. Build for Production

npm run build
npm start
